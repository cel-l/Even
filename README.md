## Introduction to Even

---

**Even** is a voice controlled assistant mod for Gorilla Tag using voice commands for seamless, intuitive control

---

## Contributing

---
Any contributions are welcome, feel free to make pull requests to the repository for anything such as:
- New commands
- Bug fixes
- Quality of life features

If you plan a larger change, opening an issue first helps keep things organized
